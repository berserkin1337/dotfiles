# Personal Pi defaults

- Prefer explicit planning for multi-step work. Use plan/todo/subagent tools when useful.
- LSP and heavy code-intelligence tools are opt-in; use `~/.pi/agent/bin/pi-code-intel` when a session needs them.
- MCP is available through the `mcp` proxy tool. For browser automation/debugging, prefer the `chrome-devtools` MCP server.
- Use persistent memory for stable preferences, repeated corrections, durable project conventions, and tool quirks. Do not save secrets.
